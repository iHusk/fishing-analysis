Concept 2 rationale

Paradigm: a cockpit-style tile dashboard where every catch field is visible and most tiles are one-tap repeat values.

How it works: tap a tile to cycle common choices; tap-hold or second tap expands the control inline, shown here on Length with quarter-inch steps.

No-scroll strategy: Tier 1 gets large tiles, Tier 2 gets medium tiles, and optional/auto fields compress into a single bottom tile row with GPS display-only.

Outside-box angle: the screen behaves like a hardware control panel for rough-water logging, with obvious homes for future BLE and voice capture.

Risks: cycling hidden values can be missed, and inline expansion needs strict height limits so a changed tile never pushes content off the phone.
